Requirements:

  • 1 Local host server supports php5.6+
  • 2 Local mysql database
  • 3 Composer installed globally local

Recommendation

Installation guides:

  • 1 Config database
  • 2 In the root directory of project, runcomposer install (Ubuntu) or php composer.phar install (Windows)
  • 3 Run php artisan serve --port=8080 to make server to run at localhost:8080
